# Ranking Algorithms Demo (WPF)

Szakdolgozathoz készített WPF alkalmazás, amely bemutatja a
- Pointwise
- Pairwise
- Listwise

Learning to Rank rangsorolási megközelítéseket.

## Funkciók
- CSV adatbetöltés
- Lekérdezés kiválasztás
- Három rangsoroló algoritmus tanítása

## Szakdolgozat
A szakdolgozat dokumentuma a dokument mappában található.
